HTML
(Lenguaje de marcado de Hipertexto)
es un lenguae muy utilisado para crear paginas wep y consiste en un conjunto de etiquetas que se utilisa para definir la estructura de un contenido de la pagina, estas etiquetas marcan un inicio y un finalen en los elementos que contiene una pagina wep como: Encabezados, Parafos, Imagenes, Enlases,etc.

Estructura Basica del Html
La estructura basica del html consta de una serie de Elementos anidados se abre con la etiqueta "html" lo cual contiene dos elementos principales: encavesado "head" el cuerpo "body".Ejemplo: 

            <html>
                <head>
                >/head>
                <body>
                    <h1>Ejemplo de titulo</h1>
                    <p>Ejemplo de parafo</p>
            >/body>
            >/html>

Etiquetas de estilos

    Lista de definiciones
  • <dl> Lista de definiciones: Esta etiqueta representa una lista descriptiva.
  • <dt>- Especifica un termino en una descripcion o lista de definiciones.
  • <dd>- Provee detalles o el significado de la definicion de un termino.Ejemplos:
    • <dl>
    <dt>Jugar</dt>

    Etiquetas para insertar imagenes y video
  • <img>- Esta etiqueta permite insertar imagenes en una pagina wep introduciendo algunos parametros.
    • "src"- Esta propiedad se usa para espesificar la ruta de la imagen cuando no se puede mostrar por algun motivo.br "alt"- Esta propiedad se utilisa para agregar un testo alternativo para la imagen cuano no se pueda mostrar.
    "width" y "height"- Esta propiedad especifica el ancho y el alto de un elemto repectivamente en pixeles.
  • <video>-Se utilisa para insertar el videos en una pagina wep como el boton de reproduccion y de pausa.
    • "controls"- Se utilisa para mostrar los controles de reproductor de video en la pagina wep como el botom de reproduccion pausa o volumen.Ejemplo:
    <img src="ruta/nombre de la imagen"width="valor px" alt="texto alternativo"> </img>
    <video src="ruta/nombre del video" controls heigh="valor px"> </video>
